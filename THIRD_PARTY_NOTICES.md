# Third-Party Notices

FilterAssets includes WebKit Content Rule List resources generated from third-party filter lists.
The exact upstream commits, included directories, output prefixes, and retrieval timestamps are recorded in `filter-sources.json` and `manifest.json`.
Each distributed version is preserved as an immutable GitHub Release with matching upstream source archives.
